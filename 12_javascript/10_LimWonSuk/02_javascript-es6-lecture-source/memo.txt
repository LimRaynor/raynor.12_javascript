1. 프로퍼티 : 객체의 상태를 나타내는 속성이에요. 예를 들어서 자동차 객체의 프로퍼티(속성)으로는 색상, 출시일, 바퀴 갯수 등이 있을 수 있죠.

2. 메소드 : 객체가 수행할 수 있는 동작이나 기능을 나타내요. 예를 들어서 자동차 객체의 메소드로는 '앞으로 가기', '정지하기', '후진하기' 등이 있을 수 있어요.



3. 차이점 : 프로퍼티는 객체의 상태(데이터)를 / 메소드는 객체의 행동(메소드)을 나타내요.

// 각각 개별 요소롤 분리하여 목록을 만드는 문법 == 배열에서 [] 을 벗기는 문법

console.log(`가장 큰 값 : ${Math.max(...arr)}`); // 어떼요 참 쉽죠? 위에껄 줄임
